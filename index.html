Sharing App via Windows Hotspot

Posted: Ankit / Date: 24 Jul 2017 / Category: Real-world used tools
✖

A program based on keras (VGG16 model) which can catogorically detect images and find fruitful output for desired info.

Based on VGG16 model and implemented in python, It categorically replaces the last layer of classifications and adds the layer as per our data requirement. This helps us and also the program not only to efficiently find the system as well save its processing computation limit for better performance. In this day and age, such a program can dramatically work for our benifit using much less of the resources.Link

✖

A sharing app inspired from the various sharing app available in the market.

How many times we had tried using various sharing apps between two windows computer and failed to send any data. Well, that problem is now solved. This software based on python socket programming, uses your shared hotspot or any shared access point to share data very rapidly. Rarely it has shown any kind of data corruption, and its accuracy matched to be 99.99% . This single program can act as sender as well as receiver as per the requirement.Link
